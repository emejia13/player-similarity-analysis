# Soccer Player Similarity Analysis
This project uses K-Means clustering to find "Value-for-Money" attackers.

## Project Overview
The goal is to identify players who match the production of elite stars but at a significantly lower market price.

## Tools Used
* Python (Pandas, NumPy, Scikit-Learn)
* Matplotlib & Seaborn for visualization
* Regular Expressions (Regex) for data cleaning

## Key Findings
* **Cluster 1 (Elites):** High cost, high production (Mbappé, Kane).
* **Cluster 2 (Value):** Low cost, high production (Lewandowski, Suárez).
